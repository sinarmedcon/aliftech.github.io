About Cananimate

Cananimate is a css animation library that is designed to be very light and easy to use. Cananimate was created with the aim of beautifying websites by adding stunning animation effects and spoiling the eyes, thus making the website more dynamic and interactive. Cananimate was created by Wahyu Krisna Aji, an IT practitioner and website developer from Pacitan, East Java, Indonesia.

<h3>Tata cara Instalasi Cananimate</h3>

Proses instalasi cananimate terbilang sangat mudah yaitu sebagai berikut :
1. Download file zip cananimate di <a href="https://github.com/aliftech/cananimate">https://github.com/aliftech/cananimate</a>
2. Ekstrak file zip cananimate sehingga akan muncul sebuah folder bernama cananimate
3. Untuk melihat nama kelas dan demo animasi yang terdapat di library cananimate silakan buka file index.html di peramban anda.
4. Setelah itu silakan kembali ke folder cananimate dan cari file cananimate.css, kemudian silakan kopi file tersebut di folder root projek kalian.
5. Silakan buka file utama pada projek website kalian dan tambahkan kode link rel="stylesheet" href="cananimate.css" pada tag head...head
6. Instalasi selesai. Selamat menikmati cananimate.

<h3> How to install Cananimate </h3>

The cananimate installation process is fairly easy, as follows:
1. Download the cananimate zip file at <a href="https://github.com/aliftech/cananimate"> https://github.com/aliftech/cananimate </a>
2. Extract the cananimate zip file so that a folder called cananimate will appear
3. To see the class name and animation demo contained in the cananimate library please open the index.html file in your browser.
4. After that please return to the cananimate folder and search for the cananimate.css file, then please copy the file in the root folder of your project.
5. Please open the main file on your website project and add the link rel = "stylesheet" href = "cananimate.css" in the head ... head tag
6. Installation complete. Enjoy the cananimate.
